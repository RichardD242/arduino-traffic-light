# arduino-traffic-light

- 1x Arduino Uno  
- 1x Breadboard  
- 3x LEDs (Rot, Gelb, Grün)  
- 3x Widerstände (220 Ω oder ähnlich) (ich hab ein kilo ohm benutzt weil es keine anderen gab)  
- Jumper-Kabel  
- USB-Kabel für den Arduino
